# CRLF*ck

## CRLF*ckとは

Brainf*ckをもとにしたプログラミング言語です。
改行だけで記述できるのが特徴です。

## 仕様

Brainf\*ckの命令を置き換えただけで、その他の仕様はBrainf\*ckのものを継承します。
命令は次のように置き換えられています。

|CRLF\*ck|Brainf\*ck|
|-|-|
|\\r\\r\\r|>|
|\\r\\r\\n|<|
|\\r\\n\\r|+|
|\\r\\n\\n|\-|
|\\n\\r\\r|\.|
|\\n\\r\\n|.|
|\\n\\n\\r|\[|
|\\n\\n\\n|\]|
